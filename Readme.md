# Lavalink on Replit

How to install and run lavalink on replit.

## Installation

Fork this github repository. then create a new replit project.

or [click me](https://replit.com/github/DarrenOfficial/lavalink-replit) to install it automatically.


### Initialisations
```bash
chmod +x start.sh
```

## Usage

```bash
./start.sh
```
